# Inbox

Dump everything here first. Don't organise while capturing — that friction kills the habit.

---

## Rules

1. **Capture fast** — one line is enough, expand later
2. **No judgement at capture time** — wrong folder is infinitely better than not captured
3. **Process weekly** — move each item to Projects, Areas, Resources, or Archives
4. **Delete freely** — if it doesn't spark action or insight on review, delete it

---

## How to Capture

**When connected to the AI agent:** Just tell me. I'll create the note and file it.

**When offline or disconnected:** Create a new note using `_Template.md`, or just write a plain `.md` file with a date prefix. The name doesn't matter — it will be processed.

**Minimum viable capture:**
```
202606171430 - quick thought about X
```

---

## Processing Checklist (weekly)

For each item in this folder:
- [ ] Does it require action? → `Projects/`
- [ ] Is it an ongoing responsibility? → `Areas/`
- [ ] Is it reference material? → `Resources/`
- [ ] Is it about a person? → `People/`
- [ ] Is it time-sensitive context? → `Journal/`
- [ ] Is it irrelevant now? → Delete
