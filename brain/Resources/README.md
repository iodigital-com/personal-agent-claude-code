# Resources

**Resources** are reference notes that outlive any single project or area. They are:
- Evergreen (not tied to a time or context)
- Re-usable (useful in multiple future situations)
- Factual or conceptual (not action-oriented)

---

## Sub-folders

| Folder | Contents |
|---|---|
| `Resources/Articles/` | Summaries and highlights from articles read |
| `Resources/Books/` | Book notes, highlights, key ideas |
| `Resources/Mental Models/` | Frameworks and thinking tools |
| `Resources/Reference/` | Facts, how-tos, reference sheets |

---

## All Resources (by tag)

_Use Obsidian's search or Dataview to query: `type: resource`_

---

## Adding a Resource

1. Choose the right sub-folder
2. Copy the relevant template from `_Templates/`
3. Set `status: evergreen` once the note is stable and will not change
4. Add to a topic MOC in `MOC/Topics/` so it's discoverable
