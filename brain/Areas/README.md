# Areas

An **area** is a sphere of activity with a standard to maintain over time — no finish line, just ongoing stewardship.

Examples: Health, Finance, Career, Learning, Home, Relationships, a specific skill domain.

## Creating an Area

1. Copy `_Templates/Area.md`
2. Save as `Areas/Area Name.md`
3. Add a row to the table above
4. Create a `MOC - Area Name.md` in `MOC/Topics/` if the area has multiple sub-notes
5. Link from `MOC/Home.md` under "Areas of Responsibility"

---

## Note on Scope

Areas often generate Projects (a health area generates a "run 5k" project) and Resources (a learning area generates reference notes). The area note is the hub that links them.
